# Skillbox

Collection of reusable AI skills.

Skillbox is a library of individual AI skills extracted from documentation, articles, open-source projects, and best practices.

Each skill represents a small, reusable capability that helps AI tools generate better code.

Examples of skills:

- architecture guidelines
- code patterns
- refactoring strategies
- testing practices
- project conventions

These skills serve as building blocks for skillsets.


Skillsets combine multiple skills into a complete workflow.

Part of the Supa Magic ecosystem.
