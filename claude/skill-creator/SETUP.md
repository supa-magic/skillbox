# Setup: skill-creator

## Prerequisites

- Claude Code CLI installed
- Git initialized in your project

## Installation

### 1. Copy skill files

Copy the following files into your project's `.claude/skills/skill/` directory:

| Source | Target |
|--------|--------|
| `claude/skill/SKILL.md` | `.claude/skills/skill/SKILL.md` |
| `claude/skill/assets/skill-template.md` | `.claude/skills/skill/assets/skill-template.md` |
| `claude/skill/create.md` | `.claude/skills/skill/create.md` |
| `claude/skill/references/skill-guidelines.md` | `.claude/skills/skill/references/skill-guidelines.md` |
| `claude/skill/references/the-complete-guide-to-building-skills-for-claude.md` | `.claude/skills/skill/references/the-complete-guide-to-building-skills-for-claude.md` |
| `claude/skill/refine.md` | `.claude/skills/skill/refine.md` |

## Verification

After installation, verify by running `/skill create my-skill` or `/skill refine my-skill` to confirm files are in place.
